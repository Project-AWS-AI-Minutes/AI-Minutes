# 기능 명세서 (AA)

## 문서 목적
AA 서비스의 핵심 기능과 우선 개발 범위를 정의한다.

## 제품 목표
- 회의 음성을 텍스트로 변환하고 핵심 내용을 요약한다.
- 회의 결과를 액션 아이템 중심으로 정리한다.
- 팀 단위 공유가 가능한 결과물을 제공한다.

## 핵심 기능
1. 회의 업로드
   - 오디오 파일 업로드
   - 기본 메타데이터(회의명, 일시, 참여자) 입력
2. 전사(Transcription)
   - 발화 단위 텍스트 변환
   - 화자 구분(가능 시)
3. 요약(Summary)
   - 전체 요약
   - 섹션별 요약(의사결정, 이슈, 할 일)
4. 액션 아이템 추출
   - 담당자, 마감일, 우선순위 추출
5. 결과 공유
   - 링크 기반 조회
   - 내보내기(Markdown/PDF)

## 비기능 요구사항
- 평균 60분 회의 기준 처리 완료 목표: 10분 이내
- 실패 시 재시도 가능해야 함
- 기본 접근 제어(로그인 사용자만 접근)

## 개발 전 확정 필요 항목
- 지원 오디오 포맷 범위
- 요약 템플릿 표준
- 액션 아이템 필수 필드 정의
