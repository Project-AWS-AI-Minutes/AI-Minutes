# 트래픽 가정 문서 (AA)

## 초기 가정
- 팀 수: 20
- 팀당 월 회의 수: 80
- 평균 회의 길이: 45분
- 동시 업로드 피크: 15건

## 처리량 추정
- 월 총 회의 수: 1,600건
- 월 총 오디오 분량: 약 72,000분
- 피크 시간 API 요청: 초당 20~40 req 가정

## 용량 계획 기준
- API 서버: 오토스케일 최소 2대
- Worker: 큐 적체 기준으로 확장
- DB: 읽기/쓰기 지연 모니터링 필수

## 개발 전 확정 필요 항목
- 실제 베타 사용자 수
- 피크 시간대 패턴
- 파일 보관 기간 정책
